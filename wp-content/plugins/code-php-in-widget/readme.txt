Many people have asked me to make a plugin which allows PHP code put inside a widget for different reasons.
That's why he believes this well done and fast plugin that lets you put any type of PHP code inside a box widget, on your front page you can see the result.

== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for 'Code PHP in Widget'
3. Activate 'Code PHP in Widget' from your Plugins page.
4. Visit 'Appearance > Widget -> 'Code PHP in Widget'.
